# TimeFlow Finance

Приложение для учёта личных финансов — по мотивам игры Time Flow.

## Возможности

- **Дашборд** — обзор чистого капитала, доходов, расходов, счетов, вкладов, брокерского портфеля
- **Доходы / Расходы** — ручные плановые записи + импортированные транзакции
- **История** — все транзакции по датам с поиском и фильтрами
- **Счета** — управление банковскими и брокерскими счетами
- **Вклады** — учёт вкладов с расчётом доходности
- **Категории** — настройка категорий доходов и расходов
- **Активы / Пассивы** — в стиле игры Rich Dad
- **Цели** — финансовые цели
- **Импорт** — из Дзен Мани (CSV) и ВТБ Брокер (XLSX)

## Стек

- **Frontend**: React + Vite + TypeScript + Tailwind CSS + shadcn/ui
- **Backend**: Express.js + in-memory storage
- **Парсинг XLSX**: xlsx

## Запуск

```bash
npm install
npm run dev
```

Откройте [http://localhost:5000](http://localhost:5000)

## Сборка

```bash
npm run build
NODE_ENV=production node dist/index.cjs
```

## Форматы импорта

### Дзен Мани CSV
- Экспорт: Настройки → Экспорт данных → CSV
- Разделитель: запятая, кодировка UTF-8 with BOM
- Первые 3 строки — метаданные, строка 4 — заголовок

### ВТБ Брокер XLSX
- Поддерживает оба формата: `Брокер_отчет` и `brokerage_report`
- Экспорт: Приложение ВТБ Мои Инвестиции → Прочее → Отчёты и справки
